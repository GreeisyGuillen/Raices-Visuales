# Raíces Visuales

## Descripción

**Raíces Visuales** es un sitio web educativo y cultural dedicado a la difusión de los mitos, leyendas y regiones folclóricas de Colombia. Su objetivo es preservar el patrimonio cultural mediante una experiencia visual e interactiva, ofreciendo información sobre personajes tradicionales, su origen, significado cultural y similitudes con otras mitologías del mundo.

---

## Características

- Página de inicio con presentación del proyecto.
- Sección de **Mitos**.
- Sección de **Leyendas**.
- Sección de **Regiones de Colombia**.
- Páginas individuales para cada mito y leyenda.
- Sistema de traducción **Español / Inglés**.
- Diseño responsivo para dispositivos móviles, tabletas y escritorio.
- Navegación intuitiva con animaciones de desplazamiento.

---

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**

---

## Funcionalidades principales

- Cambio dinámico entre español e inglés.
- Traducciones independientes para cada página.
- Contenido organizado por categorías.
- Información histórica y cultural sobre cada personaje.
- Adaptación automática a diferentes tamaños de pantalla.

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/usuario/raices-visuales.git
```

2. Abrir la carpeta del proyecto.

3. Ejecutar `inicio.html` desde un navegador web.

> También puede utilizarse la extensión **Live Server** de Visual Studio Code para una mejor experiencia.
